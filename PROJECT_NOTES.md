# Football Scouting Platform

## Objective

A platform that helps football players showcase their profiles, allows coaches to discover talent, and generates scouting reports.

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication

## Features

### Authentication

- Register
- Login
- JWT Token

### Players

- Create Player
- Get Players
- Update Player
- Delete Player

### Coaches

- Create Coach
- Get Coaches

### Trials

- Create Trial
- Get Trials

### Dashboard

- Top Scorers
- Top Assists
- Fastest Players
- Best Stamina

### AI Scout Report

- Strengths
- Weaknesses
- Rating
- Potential
- Training Advice

## Database Collections

- Users
- Players
- Coaches
- Trials
